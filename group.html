Intelligent Computing and Networking (ICN) Research Group

We do research in:
  1. Intelligent Networking: Network Designing and Optimization based on Machine Learning, Deep Learning, etc.
  2. Programmable Networks: Software Defined Networking (SDN), Network Function Virtualization (NFV)
  3. Machine Learning Systems
  4. Data Center Networking and Cloud Computing
  5. Edge Computing and Fog Computing
